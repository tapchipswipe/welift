# Florida Class B + Pre-Pilot Compliance Checklist

Complete before signing a paid HOA/CDD pilot in Manatee or Sarasota County.

## Licensing

- [ ] Confirm operating model with FDACS Division of Licensing (Class B agency required?)
- [ ] Form entity (FL LLC/Corp) and register with Sunbiz
- [ ] Apply for Class B if required; budget time for background / process
- [ ] Plan individual licenses for agents (Class D / as required)
- [ ] Identify fingerprint / licensing logistics (e.g. North Port FDACS regional office)

## Insurance (see insurance-quote-packet.md)

- [ ] Submit packet to ≥2 brokers
- [ ] Bind GL $1M/$2M (or higher if CAM requires)
- [ ] Bind security E&O / professional liability
- [ ] Bind workers comp when employee count triggers / contracts require
- [ ] Confirm assault & battery / false arrest / wrongful detention wording
- [ ] File statutory COI with FDACS (≥$300k floor — still buy higher limits)
- [ ] Prepare sample COI with additional insured + waiver language for CAM review

## Contract

- [ ] Attorney review of service-agreement-liability-clauses.md
- [ ] Post Orders template per community (guest list rules, deny/escalate)
- [ ] SLA credits (optional) approved commercially
- [ ] Data retention / video policy written

## Operations before go-live

- [ ] Install partner selected; acceptance test on cameras/audio/gate relay
- [ ] Failover: what happens if internet dies (notify list, local fail mode)
- [ ] 911 / on-call escalation tree per site
- [ ] Agent training on Post Orders + “verify don’t police”
- [ ] Soft parallel run (shadow) 3–7 nights before cutting over overnight booth

## No-go if

- [ ] Cannot bind required insurance at viable price
- [ ] Cannot obtain required license
- [ ] CAM refuses without $5M+ umbrella you cannot buy
- [ ] Hardware partner unavailable for pilot site
