# Insurance Quote Packet — Florida Virtual Gate / Remote Access Control

**Purpose:** Everything to send a broker so you can get real quotes **before** a paid HOA pilot.  
**Jurisdiction:** Florida (Class B private security agency path highly likely if you employ agents who control access / provide security services).

> Not legal advice. Confirm licensing with FDACS Division of Licensing and coverage with a licensed Florida agent.

---

## 1. Statutory floor (not enough for HOAs)

Under **Florida Statute §493.6110**, a Class “B” agency license requires filing commercial general liability of at least **$300,000** combined single limit (death, BI, PD, personal injury), with FDACS as additional insured for cancellation notices.

**HOAs and CAMs will typically demand more.** Plan for:

| Coverage | Target for HOA pilots |
|----------|----------------------|
| Commercial General Liability | **$1M per occurrence / $2M aggregate** |
| Umbrella / Excess | **$1M–$5M** (ask CAM what their master policy requires) |
| Professional Liability / Security E&O | **$1M** (wrongful admit/deny, failure-to-perform) |
| Workers’ Compensation | Statutory (FL: generally required at 4+ employees) |
| Cyber / privacy (video, guest lists, PII) | **$1M** recommended |
| Hired / Non-Owned Auto | If supervisors drive between sites |
| Assault & battery / false arrest / wrongful detention wording | Confirm **included or buy-back** — critical for access control |

Unarmed remote SOC work is usually cleaner for carriers than armed posts — keep the model **unarmed verification only**.

## 5. Licensing companion (FDACS)

Parallel track with insurance:

1. Confirm whether your operating model requires **Class B** (almost certainly if marketing as security / access control with employed agents).  
2. Individual agent licensing as required (Class D etc. — confirm current FDACS rules).  
3. File COI on **FDACS-16004** process for the statutory CGL.  
4. North Port FDACS regional office exists for fingerprinting / guard card logistics (Allied Sarasota careers pages reference North Port office) — useful locally.

**No-go:** If you cannot obtain Class B + $1M/$2M GL + E&O at a price that still leaves overnight margins positive, pause paid pilots.
