# Touch 1 — Info-gathering outreach (pre-CAM)

Use **before** the formal [cam-kickoff-email.md](cam-kickoff-email.md). Goal: name + dealer facts, not a signed deal.

---

## A) Call script — Associa Gulf Coast front desk (2 min)

```
Hi — Lucas Despot, local SWFL. I'm coordinating overnight gate attendant
routing for The Inlets at Riverdale in Bradenton. Could you tell me which
community manager handles that association, and the best email for them?

I'm not asking for a sales meeting yet — just the right contact so we don't
email the wrong person. Thank you.
```

Write down: **Name · Email · Phone · Date**.

---

**Subject:**

```
The Inlets at Riverdale — quick question on overnight gate / dealer contact

```

Lucas Despot
We Lift
