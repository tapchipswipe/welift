## 1) Form FL LLC (Sunbiz) — ~$125 · 1–3 business days

### Steps

1. Go to [dos.fl.gov/sunbiz](https://dos.fl.gov/sunbiz/) → Articles of Organization (LLC).
2. Name search: try **We Lift LLC**, **WeLift LLC**, **We Lift Security LLC** (pick available).
3. Registered agent: Lucas (home address) or registered-agent service.
4. Principal address: SWFL address you control.
5. Pay filing fee online; save PDF receipt + document number.
6. Same day: get **EIN** free at [irs.gov](https://www.irs.gov/businesses/small-businesses-self-employed/apply-for-an-employer-identification-number-ein-online) (SSN + LLC info).
7. Open business checking when EIN arrives (optional but helps brokers).

## 3) FDACS Class B — call script (15 min)

**Who:** FDACS Division of Licensing — Florida private security licensing line (confirm current number on [fdacs.gov](https://www.fdacs.gov/) → Licensing → Private Security).

```
Hi — I'm starting a Florida company that does unarmed remote visitor
verification at HOA gates and remotely unlocks gates per the association's
post orders. No armed work, no patrol, no detention.

Does this require a Class B private security agency license?
If yes, what's the application path and typical timeline?
Is subcontracting under an existing Class B agency an acceptable path
for a first pilot?

Company name: We Lift [LLC]. Location: Manatee/Sarasota.
```

### Decision tree

| FDACS answer | Your path |
|--------------|-----------|
| Class B required | Apply **or** subcontract under licensed agency before paid opens |
| Gray / tech vendor | Get **written** guidance; still buy E&O; soft pilot only until clear |
| Subcontract OK | Prefer for speed — licensee on contract/COI |

Log answer in [decision-log.md](decision-log.md) and [compliance-path.md](compliance-path.md).
